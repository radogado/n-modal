Options

  • Open a modal window with any content using 
    nui.componentModal.openFullWindow("Any content");
  • Modals with 
    id
    matching the URI hash are opened automatically 
  • #content
    is recommended to avoid HTML head parts 
  • nui.componentModal.openFullWindow(HTML, "0% { … } 100% { … }")
    for custom animation, automatically reversed on close 
  • a.n-modal-link[data-anim]
    parameter specifies custom animation 
  • a.n-modal-link.n-modal--full
    for full width/height modal window 
  • a.n-modal-link.n-modal--rounded
